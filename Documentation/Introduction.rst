============
Introduction
============

The **Digital Page Flip** extension provides an interactive flipbook viewer
for TYPO3 websites. It converts uploaded PDF files into browsable online
catalogs with realistic page-turning effects.

What it does
============

1. An editor uploads a PDF in the TYPO3 backend.
2. The extension automatically converts each page to optimized WebP images
   using Ghostscript and ImageMagick.
3. In the frontend, the page-flip library renders the images as an
   interactive catalog that users can browse by clicking, swiping or
   using keyboard arrows.

Who is it for?
==============

- **Editors** who want to publish PDF flyers, brochures or catalogs as
  interactive online versions without manual image preparation.
- **Agencies** building TYPO3 websites for retail, gastronomy or any
  business that distributes printed flyers.
- **Developers** looking for a ready-made, extensible flipbook solution
  that integrates cleanly with TYPO3's FAL and Extbase architecture.
